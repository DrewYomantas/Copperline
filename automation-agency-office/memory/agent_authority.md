# Agent Authority Policy

## Purpose
Define what agents can do autonomously versus what requires founder review/approval.

## Agents May Do Autonomously
- Organize internal documentation and update memory files.
- Conduct prospect/company research from public sources.
- Draft internal summaries, outreach drafts, and proposal drafts.
- Update CRM/pipeline records with factual status notes.
- Generate task lists, checklists, and draft workflows.

## Founder Review Required Before Execution
- Any external message draft intended for prospects/clients.
- Final pricing recommendations and proposal documents.
- Scope changes that alter timeline, deliverables, or cost.
- Content posts representing company claims or outcomes.

## Founder Approval Required
- Sending any outbound message to external parties.
- Final contract/proposal issuance.
- Price exceptions outside defined ranges.
- Tool access changes involving credentials or sensitive systems.
